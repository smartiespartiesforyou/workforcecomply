# WorkforceComply 🏛️

**WorkforceComply** is a professional compliance automation tool designed for Louisiana healthcare HR departments. It streamlines the verification process for LDH Adverse Actions and CNA licensing status.

## 🚀 Key Features
* **Point-in-Time Lookups:** Instant checks against the Louisiana CNA Registry and LDH Adverse Action databases.
* **Audit-Ready Documentation:** Automatically generates dated, source-labeled PDF records for every check.
* **Batch Processing:** Support for standardized Excel/CSV uploads to screen entire rosters in seconds (ideal for UKG Ready users).
* **Data Security:** Strict 5-day retention policy to ensure privacy—we are a verification tool, not a system of record.

## 📋 How It Works
1. **Upload:** HR uploads a standardized Excel file containing Name and SSN.
2. **Verify:** The engine performs real-time lookups across state registries.
3. **Download:** Results are compiled into PDFs available for a limited 5-day window.
